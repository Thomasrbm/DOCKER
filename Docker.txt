Dockerfile : Dit comment construire l image. (snap comme vm)





FROM php:8.1-cli   => use version de php.

WORKDIR /app => dossier app sera espace de travail (dossier ou place le code et execute)

COPY . /app => copy le code depuis le dossier vers app (prend code du dossier courrant et le copie dans app)

RUN docker-php-ext-install pdo pdo_mysql => install les extensions 

CMD ["php", "-S", "0.0.0.0:8000", "-t", "public"] => lance un serveur php integres 

CMD = cmd par defaut au moment du lancement
php = lance un serveur php integre avec le -S 
-S = -S : option de PHP qui signifie “Start built‑in web server” (démarrer le serveur web intégré).
c est en local host pck c est docker.
0.0.0.0  = Ecoute toutes les interfaces : peut se co depuis une autre machine du reseau.
8000 = port de mon pc
-t = defini la racine web (fichier ou le serveur va se servir)
public = le nom de ce fichier ( y aura fichier accessible au navigateur : index.php, CSS JS, images, )
